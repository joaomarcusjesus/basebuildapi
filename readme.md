## What am I using?

* Laravel 5.6
* Vue JS
* Passport
* A few libs from the amazing spatie (https://github.com/spatie)

## Features

* Laravel slug
* Laravel resources
* Laravel custom request
* Laravel policies
* Laravel custom pagination

## Show me routes
* php artisan route:list

## Build me

*  npm install
*  composer install
*  php artisan migrate --seed
